* Requires Linux

  * ``` python3 ```
  
  * ``` pygobject3 ```
 
  * ``` python3-gobject ```

  
  
* Install On Linux

  * ``` cd && git clone https://github.com/yucefsourani/PygiHashLibGui ```

  * ``` cd ~/PygiHashLibGui ```

  * ``` sudo ./setup.py install ```




* Requires Windows

  * ``` Visual C++ Redistributable ```


* Install On Windows

  * <https://github.com/yucefsourani/PygiHashLibGui/raw/master/windows/pygihashlibgui.zip>
